# 二维码图片使用说明

## 文件位置
请将您的微信群二维码图片放置在以下位置：
```
images/qr-wechat-group.png
```

## 图片规格要求
- **推荐尺寸**：200x200px 或 256x256px
- **文件格式**：PNG（推荐）或 JPG
- **文件大小**：建议小于 50KB
- **背景**：建议白色或透明背景

## 上传步骤
1. 准备您的微信群二维码截图
2. 确保图片清晰可扫描
3. 重命名为 `qr-wechat-group.png`
4. 放置在 `images/` 文件夹中

## 备用方案
如果暂时没有二维码图片，可以：
1. 使用在线二维码生成器创建临时二维码
2. 将微信群的链接转换为二维码
3. 或者保留当前的占位符样式

## 注意事项
- 确保二维码图片清晰，便于用户扫描
- 定期检查二维码是否有效
- 如有多个群聊，考虑使用合并二维码

## 文件结构
```
importNames/

│   └── qr-wechat-group.png  <- 请将二维码图片放置在此处
```
